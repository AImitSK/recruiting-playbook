KI-Addon für Recruiting Playbook – KI-gestützte Stellenanzeigen, Textverbesserung und CV-Matching.

== Description ==

Das KI-Addon erweitert Recruiting Playbook um leistungsstarke KI-Funktionen:

* **CV-Matching:** Automatischer Abgleich von Lebensläufen mit Stellenanforderungen
* **Job-Finder:** KI-gestützte Stellensuche für Bewerber
* **Textverbesserung:** KI-optimierte Stellenanzeigen

Dieses Addon erfordert das Plugin [Recruiting Playbook](https://recruiting-playbook.com/).

== Installation ==

1. Recruiting Playbook installieren und aktivieren
2. Recruiting Playbook KI installieren und aktivieren
3. Lizenz unter Einstellungen → Recruiting Playbook → KI-Analyse aktivieren
