Recruiting Playbook stellt neun Shortcodes bereit, mit denen Sie Stellenanzeigen, Bewerbungsformulare und KI-Features in jede WordPress-Seite einbinden können.

<Callout type="info">
Alternativ zu Shortcodes können Sie die nativen [Gutenberg-Blöcke](/docs/gutenberg-blocks) verwenden, die eine visuelle Bearbeitung ermöglichen. Für Avada-Nutzer stehen zusätzlich native [Avada/Fusion Builder Elemente](/docs/avada-integration) zur Verfügung.
</Callout>

## rp_jobs

Zeigt eine Liste der Stellenanzeigen an.

![Stellenliste im Grid-Layout](/screenshots/sc-job-liste.png)

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `limit` | Anzahl der angezeigten Stellen | `10` |
| `category` | Filter nach Kategorie-Slug (mehrere mit Komma) | - |
| `location` | Filter nach Standort-Slug (mehrere mit Komma) | - |
| `type` | Filter nach Beschäftigungsart | - |
| `featured` | Nur hervorgehobene Stellen (`true`/`false`) | `false` |
| `orderby` | Sortierung (`date`, `title`, `rand`) | `date` |
| `order` | Reihenfolge: `ASC` oder `DESC` | `DESC` |
| `columns` | Spalten im Grid (1-4, leer = Design-Einstellung) | - |

### Beispiele

**Einfache Stellenliste (5 neueste):**

```html
[rp_jobs limit="5"]
```

**Gefiltert nach Kategorie, 2-spaltig:**

```html
[rp_jobs limit="5" category="it" columns="2"]
```

**Pflege-Stellen in Berlin:**

```html
[rp_jobs category="pflege" location="berlin" columns="2"]
```

**Nur Featured Jobs, zufällige Reihenfolge:**

```html
[rp_jobs featured="true" limit="3" orderby="rand"]
```

## rp_job_search

Zeigt ein Suchformular mit Filtern und Ergebnisliste. Ideal für die Karriereseite.

![Jobsuche mit Filtern](/screenshots/sc-jobfinder.png)

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `show_search` | Suchfeld anzeigen | `true` |
| `show_category` | Kategorie-Filter anzeigen | `true` |
| `show_location` | Standort-Filter anzeigen | `true` |
| `show_type` | Beschäftigungsart-Filter anzeigen | `true` |
| `limit` | Stellen pro Seite | `10` |
| `columns` | Spalten im Grid | `1` |

### Beispiele

**Vollständige Jobsuche:**

```html
[rp_job_search limit="10" columns="2"]
```

**Ohne Beschäftigungsart-Filter:**

```html
[rp_job_search show_type="false" limit="20"]
```

**Nur Suchfeld, keine Dropdowns:**

```html
[rp_job_search show_category="false" show_location="false" show_type="false"]
```

## rp_job_count

Zeigt die Anzahl der verfügbaren Stellen an. Ideal für Headlines.

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `category` | Filter nach Kategorie-Slug | - |
| `location` | Filter nach Standort-Slug | - |
| `type` | Filter nach Beschäftigungsart | - |
| `format` | Text für Mehrzahl mit Platzhalter | Siehe Beispiele |
| `singular` | Text für Einzahl | Siehe Beispiele |
| `zero` | Text für 0 Stellen | `Keine offenen Stellen` |

### Beispiele

**Standard-Zähler:**

```html
Aktuell [rp_job_count] verfügbar.
```

**Gefiltert nach Kategorie:**

```html
[rp_job_count category="it" format="{count} IT-Stellen"]
```

**Eigene Texte:**

```html
[rp_job_count format="Wir haben {count} Jobs" singular="Wir haben {count} Job" zero="Aktuell keine Jobs"]
```

## rp_featured_jobs

Zeigt hervorgehobene (Featured) Stellen an.

![Featured Jobs im Grid](/screenshots/sc-top-job.png)

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `limit` | Anzahl der Stellen | `3` |
| `columns` | Spalten im Grid (1-4) | `3` |
| `title` | Optionale Überschrift | - |
| `show_excerpt` | Auszug anzeigen | `true` |

### Beispiele

**Standard Featured Jobs:**

```html
[rp_featured_jobs]
```

**Mit Überschrift, 2 Spalten:**

```html
[rp_featured_jobs title="Top-Stellenangebote" columns="2" limit="4"]
```

## rp_latest_jobs

Zeigt die neuesten Stellenanzeigen an. Ideal für Sidebars.

![Neueste Stellen im Grid](/screenshots/sc-neu.png)

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `limit` | Anzahl der Stellen | `5` |
| `columns` | Spalten (0 = Liste, 1-4) | `0` |
| `title` | Optionale Überschrift | - |
| `category` | Filter nach Kategorie-Slug | - |
| `show_date` | Datum anzeigen | `true` |
| `show_excerpt` | Auszug anzeigen | `false` |

### Beispiele

**Kompakte Liste für Sidebar:**

```html
[rp_latest_jobs limit="5"]
```

**Mit Überschrift und Grid:**

```html
[rp_latest_jobs title="Neueste Stellen" columns="2" limit="6" show_excerpt="true"]
```

## rp_job_categories

Zeigt alle Job-Kategorien als klickbare Karten an.

![Kategorie-Karten mit Stellenanzahl](/screenshots/sc-kategorie.png)

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `columns` | Spalten im Grid (1-6) | `4` |
| `show_count` | Anzahl pro Kategorie anzeigen | `true` |
| `hide_empty` | Leere Kategorien verstecken | `true` |
| `orderby` | Sortierung (`name`, `count`) | `name` |

### Beispiele

**Standard Kategorie-Übersicht:**

```html
[rp_job_categories]
```

**3 Spalten, sortiert nach Anzahl:**

```html
[rp_job_categories columns="3" orderby="count"]
```

**Alle Kategorien (auch leere):**

```html
[rp_job_categories hide_empty="false"]
```

## rp_application_form

Zeigt das Bewerbungsformular an. Wird normalerweise automatisch auf der Stellendetail-Seite eingebunden, kann aber auch manuell platziert werden.

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `job_id` | ID der Stelle | automatisch |
| `title` | Überschrift des Formulars | `Jetzt bewerben` |
| `show_job_title` | Stellentitel über dem Formular anzeigen | `true` |
| `show_progress` | Fortschrittsanzeige anzeigen | `true` |

### Beispiele

**Standard (auf Stellendetail-Seite):**

```html
[rp_application_form]
```

**Für eine bestimmte Stelle:**

```html
[rp_application_form job_id="123" title="Ihre Bewerbung"]
```

<Callout type="info">
Wenn Sie `job_id` nicht angeben, versucht das Plugin automatisch die aktuelle Stelle zu erkennen. Auf einer Nicht-Stellen-Seite ist die Angabe der `job_id` erforderlich.
</Callout>

## rp_ai_job_finder

KI-gestützter Job-Finder: Besucher laden ihren Lebenslauf hoch und erhalten passende Stellenvorschläge.

<Callout type="warning">
Dieses Feature erfordert das AI-Addon.
</Callout>

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `title` | Überschrift | `Finde deinen Traumjob` |
| `subtitle` | Erklärungstext | `Lade deinen Lebenslauf hoch...` |
| `limit` | Max. Anzahl Vorschläge (1-10) | `5` |
| `class` | Zusätzliche CSS-Klassen | - |

### Beispiel

```html
[rp_ai_job_finder title="Welcher Job passt zu dir?" limit="3"]
```

## rp_ai_job_match

"Passe ich zu diesem Job?" Button mit KI-Analyse.

<Callout type="warning">
Dieses Feature erfordert das AI-Addon.
</Callout>

### Attribute

| Attribut | Beschreibung | Standard |
|----------|-------------|----------|
| `job_id` | ID der Stelle | automatisch |
| `title` | Button-Text | `Passe ich zu diesem Job?` |
| `style` | Button-Variante (`outline`) | - |
| `class` | Zusätzliche CSS-Klassen | - |

### Beispiele

**Standard (auf Stellendetail-Seite):**

```html
[rp_ai_job_match]
```

**Mit eigenem Text und Outline-Style:**

```html
[rp_ai_job_match title="Bin ich geeignet?" style="outline"]
```

## Shortcodes im Block-Editor

Im WordPress Block-Editor (Gutenberg) können Sie Shortcodes über den **Shortcode-Block** einfügen:

1. Neuen Block hinzufügen
2. Nach "Shortcode" suchen
3. Shortcode einfügen, z.B. `[rp_job_search]`

<Callout type="info">
Für eine bessere Benutzererfahrung empfehlen wir die nativen [Gutenberg-Blöcke](/docs/gutenberg-blocks), die eine visuelle Vorschau und einfache Einstellungen bieten.
</Callout>
