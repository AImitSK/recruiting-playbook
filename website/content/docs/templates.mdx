---
title: Templates
description: So ueberschreiben Sie die Plugin-Templates in Ihrem Theme.
---

Recruiting Playbook verwendet PHP-Templates fuer die Darstellung von Stellenanzeigen im Frontend. Diese Templates koennen in Ihrem Theme ueberschrieben werden, um das Aussehen an Ihr Design anzupassen.

## Template-Ueberschreibung

### Schritt 1: Ordner erstellen

Erstellen Sie im Root-Verzeichnis Ihres Themes (oder Child-Themes) den Ordner `recruiting-playbook/`:

```bash
wp-content/themes/ihr-theme/recruiting-playbook/
```

### Schritt 2: Templates kopieren

Kopieren Sie die gewuenschten Templates aus dem Plugin-Ordner:

```bash
# Quelle (Plugin)
wp-content/plugins/recruiting-playbook/templates/

# Ziel (Theme)
wp-content/themes/ihr-theme/recruiting-playbook/
```

### Schritt 3: Anpassen

Bearbeiten Sie die kopierten Dateien nach Ihren Wuenschen. Das Plugin laedt automatisch die Theme-Version, wenn sie vorhanden ist.

## Verfuegbare Templates

| Template | Beschreibung |
|----------|-------------|
| `archive-job_listing.php` | Stellenliste (Archiv-Seite) |
| `single-job_listing.php` | Einzelne Stellenanzeige |
| `partials/job-card.php` | Karte fuer eine Stelle in der Liste |
| `partials/job-filters.php` | Filterleiste der Jobsuche |
| `partials/application-form.php` | Bewerbungsformular |
| `partials/application-success.php` | Erfolgsseite nach Bewerbung |

## Template-Hierarchie

Das Plugin sucht Templates in dieser Reihenfolge:

1. `wp-content/themes/ihr-child-theme/recruiting-playbook/`
2. `wp-content/themes/ihr-parent-theme/recruiting-playbook/`
3. `wp-content/plugins/recruiting-playbook/templates/`

<Callout type="tip" title="Child-Theme nutzen">
Verwenden Sie immer ein Child-Theme fuer Template-Ueberschreibungen. So bleiben Ihre Anpassungen bei Theme-Updates erhalten.
</Callout>

## Beispiel: Stellenkarte anpassen

Kopieren Sie `partials/job-card.php` in Ihr Theme und passen Sie das HTML an:

```php
<?php
// wp-content/themes/ihr-theme/recruiting-playbook/partials/job-card.php

$job_id    = get_the_ID();
$location  = get_the_terms( $job_id, 'job_location' );
$type      = get_the_terms( $job_id, 'employment_type' );
?>
<div class="my-job-card">
    <h3>
        <a href="<?php the_permalink(); ?>">
            <?php the_title(); ?>
        </a>
    </h3>
    <?php if ( $location ) : ?>
        <span class="my-job-location">
            <?php echo esc_html( $location[0]->name ); ?>
        </span>
    <?php endif; ?>
    <?php if ( $type ) : ?>
        <span class="my-job-type">
            <?php echo esc_html( $type[0]->name ); ?>
        </span>
    <?php endif; ?>
    <?php the_excerpt(); ?>
</div>
```

## E-Mail-Templates

E-Mail-Templates koennen ebenfalls ueberschrieben werden:

| Template | Zweck |
|----------|------|
| `emails/application-received.php` | Eingangsbestaetigung fuer Bewerber |
| `emails/application-rejected.php` | Absage an Bewerber |
| `emails/interview-invitation.php` | Einladung zum Vorstellungsgespraech |

<Callout type="warning">
Aendern Sie in E-Mail-Templates keine Platzhalter-Variablen (z.B. `{{applicant_name}}`). Diese werden vom Plugin zur Laufzeit ersetzt.
</Callout>
