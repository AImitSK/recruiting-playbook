Recruiting Playbook bietet native Gutenberg-Bloecke fuer alle wichtigen Funktionen. Die Bloecke sind kostenlos verfuegbar und bieten eine komfortable Alternative zu Shortcodes.

## Vorteile der Gutenberg-Bloecke

- **Visuelles Editing** direkt im Block-Editor
- **Einfache Bedienung** per Drag & Drop
- **Einstellungen** ueber die Sidebar (Inspector Controls)
- **Block-Patterns** fuer haeufige Layouts
- **Theme-Integration** mit automatischem Styling

## Verfuegbare Bloecke

| Block | Beschreibung | Feature |
|-------|--------------|---------|
| Stellenliste | Liste der Stellenanzeigen | Free |
| Stellensuche | Suchformular mit Filtern | Free |
| Stellen-Zaehler | Dynamischer Zaehler fuer Headlines | Free |
| Featured Jobs | Hervorgehobene Stellen | Free |
| Neueste Stellen | Aktuellste Stellenanzeigen | Free |
| Job-Kategorien | Klickbare Kategorie-Karten | Free |
| Bewerbungsformular | Mehrstufiges Bewerbungsformular | Free |
| KI-Job-Finder | Lebenslauf-Upload mit KI-Matching | AI-Addon |
| KI-Job-Match | "Passe ich zu diesem Job?" Button | AI-Addon |

## Stellenliste (rp/jobs)

Der Hauptblock zur Anzeige von Stellenanzeigen.

### Einstellungen

| Einstellung | Beschreibung | Standard |
|-------------|--------------|----------|
| Anzahl Stellen | Wie viele Stellen angezeigt werden | `10` |
| Spalten | Grid-Spalten (1-4) | `2` |
| Kategorie | Filter nach Kategorie | - |
| Standort | Filter nach Standort | - |
| Beschaeftigungsart | Filter nach Art | - |
| Nur Featured | Nur hervorgehobene Stellen | Nein |
| Sortieren nach | Datum, Titel oder Zufall | Datum |
| Reihenfolge | Aufsteigend oder Absteigend | Absteigend |

### Beispiel

1. Block hinzufuegen (+ Symbol)
2. Nach "Stellenliste" suchen
3. Block einfuegen
4. In der Sidebar Einstellungen anpassen

## Stellensuche (rp/job-search)

Suchformular mit Filtern und Live-Ergebnissen.

### Einstellungen

| Einstellung | Beschreibung | Standard |
|-------------|--------------|----------|
| Suchfeld anzeigen | Freitext-Suche aktivieren | Ja |
| Kategorie-Filter | Dropdown fuer Kategorien | Ja |
| Standort-Filter | Dropdown fuer Standorte | Ja |
| Beschaeftigungsart-Filter | Dropdown fuer Arten | Ja |
| Stellen pro Seite | Anzahl der Ergebnisse | `10` |
| Spalten | Grid-Spalten | `1` |

## Stellen-Zaehler (rp/job-count)

Dynamischer Zaehler fuer Headlines wie "Aktuell 12 offene Stellen".

### Einstellungen

| Einstellung | Beschreibung | Standard |
|-------------|--------------|----------|
| Kategorie | Filter nach Kategorie | - |
| Standort | Filter nach Standort | - |
| Beschaeftigungsart | Filter nach Art | - |
| Format (Mehrzahl) | Text mit {count} Platzhalter | "{count} offene Stellen" |
| Format (Einzahl) | Text fuer 1 Stelle | "{count} offene Stelle" |
| Format (Null) | Text fuer 0 Stellen | "Keine offenen Stellen" |

### Beispiel

Der Zaehler kann in Ueberschriften eingefuegt werden:

```
Willkommen! Aktuell [Stellen-Zaehler] bei uns.
```

Ergebnis: "Willkommen! Aktuell 12 offene Stellen bei uns."

Zeigt die neuesten Stellenanzeigen an - ideal fuer Sidebars.

### Einstellungen

| Einstellung | Beschreibung | Standard |
|-------------|--------------|----------|
| Anzahl | Anzahl der Stellen | `5` |
| Spalten | Grid-Spalten (0 = Liste) | `0` |
| Ueberschrift | Optionale Ueberschrift | - |
| Kategorie | Filter nach Kategorie | - |

## Job-Kategorien (rp/job-categories)

Zeigt alle Kategorien als klickbare Karten an.

### Einstellungen

| Einstellung | Beschreibung | Standard |
|-------------|--------------|----------|
| Spalten | Grid-Spalten (1-6) | `4` |
| Anzahl anzeigen | Jobs pro Kategorie anzeigen | Ja |
| Leere verstecken | Kategorien ohne Jobs ausblenden | Ja |
| Sortierung | Nach Name oder Anzahl | Name |

## Bewerbungsformular (rp/application-form)

Mehrstufiges Bewerbungsformular mit Datei-Upload.

### Einstellungen

| Einstellung | Beschreibung | Standard |
|-------------|--------------|----------|
| Stellen-ID | Fuer welche Stelle (0 = automatisch) | `0` |
| Ueberschrift | Formular-Ueberschrift | "Jetzt bewerben" |
| Stellentitel anzeigen | Name der Stelle anzeigen | Ja |
| Fortschritt anzeigen | Schritt-Anzeige aktivieren | Ja |

<Callout type="info">
Auf Stellendetail-Seiten wird die Stelle automatisch erkannt. Auf anderen Seiten muss die Stellen-ID angegeben werden.
</Callout>

| Einstellung | Beschreibung | Standard |

| Ueberschrift | Titel des Bereichs | "Finde deinen Traumjob" |
| Untertitel | Erklaerungstext | - |
| Max. Vorschlaege | Anzahl der Empfehlungen (1-10) | `5` |

## Block-Patterns

Recruiting Playbook stellt vorgefertigte Block-Patterns bereit:

### Karriereseite

Komplettes Layout mit Ueberschrift, Zaehler, Featured Jobs und Suche.

### Stellen-Sidebar

Kompakte Liste der neuesten Stellen fuer Sidebars.

### Job-Finder Hero

Hero-Bereich mit KI-Job-Finder (erfordert AI-Addon).

## Bloecke einfuegen

1. Im Block-Editor auf das **+** Symbol klicken
2. Nach "Recruiting" oder dem Block-Namen suchen
3. Block auswaehlen und einfuegen
4. Einstellungen in der rechten Sidebar anpassen

<Callout type="info">
Die Bloecke befinden sich in der Kategorie "Recruiting Playbook" im Block-Inserter.
</Callout>

## Design & Branding

Die Bloecke uebernehmen automatisch die Einstellungen aus dem Design & Branding Bereich:

- Farben (Primaerfarbe, Buttons)
- Typografie (Schriftgroessen)
- Card-Layouts (Standard, Kompakt, Featured)
- Grid-Einstellungen

Individuelle Anpassungen koennen pro Block in der Sidebar vorgenommen werden.
