title: Gutenberg Bloecke
description: Native WordPress-Bloecke fuer den Block-Editor mit visueller Vorschau und InspectorControls.

Recruiting Playbook bietet native Gutenberg-Bloecke als Alternative zu Shortcodes. Die Bloecke sind fuer alle Nutzer verfuegbar (Free & Pro) und bieten eine intuitive Bedienung direkt im Block-Editor.

<Callout type="tip">
**Empfehlung:** Wenn Sie den Block-Editor (Gutenberg) verwenden, sind die nativen Bloecke die beste Wahl. Sie bieten eine visuelle Vorschau und komfortable Einstellungen in der Seitenleiste.
</Callout>

## Verfuegbare Bloecke

| Block | Beschreibung | Shortcode-Aequivalent |
|-------|-------------|----------------------|
| **Stellenliste** | Zeigt Stellenanzeigen als Grid oder Liste | `[rp_jobs]` |
| **Stellensuche** | Suchformular mit Filtern und Ergebnissen | `[rp_job_search]` |
| **Stellen-Zaehler** | Zeigt die Anzahl offener Stellen | `[rp_job_count]` |
| **Featured Jobs** | Hervorgehobene Stellenanzeigen | `[rp_featured_jobs]` |
| **Neueste Stellen** | Die zuletzt veroeffentlichten Stellen | `[rp_latest_jobs]` |
| **Job-Kategorien** | Alle Kategorien als klickbare Karten | `[rp_job_categories]` |
| **Bewerbungsformular** | Formular fuer Bewerbungen | `[rp_application_form]` |
| **KI-Job-Finder** | Lebenslauf-Upload mit KI-Matching | `[rp_ai_job_finder]` |
| **KI-Job-Match** | Button fuer KI-gestütztes Matching | - |

---

## Bloecke einfuegen

1. Klicken Sie auf das **+** Symbol im Editor
2. Suchen Sie nach "Recruiting" oder dem Block-Namen
3. Klicken Sie auf den gewuenschten Block
4. Konfigurieren Sie die Einstellungen in der **Seitenleiste** (rechts)

Alle Bloecke befinden sich in der Kategorie **Recruiting Playbook**.

---

## Stellenliste

Zeigt eine konfigurierbare Liste von Stellenanzeigen an.

### Einstellungen

**Anzeige:**
- **Anzahl Stellen** - Maximale Anzahl (1-50)
- **Spalten** - Grid-Layout (1-4 Spalten)

**Filter:**
- **Kategorie** - Nur Stellen einer bestimmten Kategorie
- **Standort** - Nur Stellen an einem bestimmten Standort
- **Beschaeftigungsart** - Vollzeit, Teilzeit, etc.
- **Nur Featured Jobs** - Nur hervorgehobene Stellen anzeigen

**Sortierung:**
- **Sortieren nach** - Datum, Titel oder Zufall
- **Reihenfolge** - Aufsteigend oder Absteigend

### Block-Attribute

| Attribut | Typ | Standard |
|----------|-----|----------|
| `limit` | number | 10 |
| `columns` | number | 2 |
| `category` | string | "" |
| `location` | string | "" |
| `type` | string | "" |
| `featured` | boolean | false |
| `orderby` | string | "date" |
| `order` | string | "DESC" |

## Stellensuche

Vollstaendiges Suchformular mit Filtern und dynamischer Ergebnisliste.

### Einstellungen

**Suchformular:**
- **Suchfeld anzeigen** - Freitext-Suche aktivieren
- **Kategorie-Filter** - Dropdown fuer Kategorien
- **Standort-Filter** - Dropdown fuer Standorte
- **Beschaeftigungsart-Filter** - Dropdown fuer Beschaeftigungsart

**Ergebnisse:**
- **Stellen pro Seite** - Anzahl pro Seite (1-50)
- **Spalten** - Grid-Layout fuer Ergebnisse

### Block-Attribute

| Attribut | Typ | Standard |
|----------|-----|----------|
| `showSearch` | boolean | true |
| `showCategory` | boolean | true |
| `showLocation` | boolean | true |
| `showType` | boolean | true |
| `limit` | number | 10 |
| `columns` | number | 1 |

---

## Stellen-Zaehler

Zeigt die Anzahl der verfuegbaren Stellen an - ideal fuer Headlines.

### Einstellungen

**Filter:**
- **Kategorie** - Nur Stellen einer Kategorie zaehlen
- **Standort** - Nur Stellen eines Standorts zaehlen
- **Beschaeftigungsart** - Nur bestimmte Beschaeftigungsart

**Textformat:**
- **Mehrere Stellen** - Text mit `{count}` Platzhalter
- **Eine Stelle** - Text fuer genau 1 Stelle
- **Keine Stellen** - Text fuer 0 Stellen

### Block-Attribute

| Attribut | Typ | Standard |
|----------|-----|----------|
| `category` | string | "" |
| `location` | string | "" |
| `type` | string | "" |
| `format` | string | "{count} offene Stellen" |
| `singular` | string | "{count} offene Stelle" |
| `zero` | string | "Keine offenen Stellen" |

## Job-Kategorien

Zeigt alle Job-Kategorien als klickbare Karten-Uebersicht.

### Einstellungen

**Anzeige:**
- **Spalten** - Grid-Layout (1-6 Spalten)
- **Anzahl Stellen anzeigen** - Badge mit Stellenanzahl
- **Leere Kategorien ausblenden** - Kategorien ohne Stellen verstecken

**Sortierung:**
- **Sortieren nach** - Name (A-Z) oder Anzahl Stellen

### Block-Attribute

| Attribut | Typ | Standard |
|----------|-----|----------|
| `columns` | number | 4 |
| `showCount` | boolean | true |
| `hideEmpty` | boolean | true |
| `orderby` | string | "name" |

## Bewerbungsformular

Zeigt das Bewerbungsformular fuer eine Stelle an.

### Einstellungen

- **Job-ID** - ID der Stelle (leer = automatische Erkennung)
- **Ueberschrift** - Titel des Formulars
- **Stellentitel anzeigen** - "Bewerbung fuer: [Titel]" anzeigen
- **Fortschrittsanzeige** - Progress-Bar bei Multi-Step-Formularen

### Block-Attribute

| Attribut | Typ | Standard |
|----------|-----|----------|
| `jobId` | number | 0 |
| `title` | string | "Jetzt bewerben" |
| `showJobTitle` | boolean | true |
| `showProgress` | boolean | true |

<Callout type="info">
Auf Stellendetail-Seiten wird die Job-ID automatisch erkannt. Auf anderen Seiten muss die ID manuell angegeben werden.
</Callout>

<Callout type="tip">
**Pro-Feature:** Mit dem Form Builder koennen Sie das Bewerbungsformular individuell gestalten - eigene Felder, Steps und Layouts.
</Callout>

## Block-Patterns

Recruiting Playbook stellt vordefinierte Block-Patterns bereit, die mehrere Bloecke kombinieren:

- **Karriereseite Komplett** - Suchformular + Kategorie-Grid + Featured Jobs
- **Stellenliste mit Filter** - Kategorie-Grid + Stellenliste
- **Job-Finder Hero** - KI-Job-Finder mit Stellen-Zaehler
- **Featured Section** - Featured Jobs als hervorgehobener Bereich
- **Bewerbungs-CTA** - Bewerbungsformular mit Ueberschrift

### Patterns einfuegen

1. Klicken Sie auf das **+** Symbol im Editor
2. Wechseln Sie zum Tab **Patterns**
3. Waehlen Sie die Kategorie **Recruiting Playbook**
4. Klicken Sie auf das gewuenschte Pattern

## Design & Branding

Alle Bloecke respektieren die Design & Branding Einstellungen (Pro-Feature):

- **Primaerfarbe** fuer Buttons und Akzente
- **Card-Design** (Kompakt/Standard/Grosszuegig)
- **Button-Stil** vom Theme oder individuell
- **Badge-Farben** fuer Tags und Kategorien

Die Einstellungen finden Sie unter **Recruiting Playbook > Einstellungen > Design & Branding**.

## Gutenberg vs. Shortcodes

| Merkmal | Gutenberg Bloecke | Shortcodes |
|---------|------------------|------------|
| Visuelle Vorschau | Ja (Placeholder) | Nein |
| Einstellungen | Seitenleiste | Code-Attribute |
| Block-Patterns | Ja | Nein |
| Theme-Integration | Ja (spacing, color) | Eingeschraenkt |
| Page Builder | Nein | Ja (Avada, etc.) |
| Classic Editor | Nein | Ja |

<Callout type="info">
Verwenden Sie **Shortcodes**, wenn Sie den Classic Editor oder Page Builder wie Avada nutzen. Fuer den Block-Editor (Gutenberg) empfehlen wir die **nativen Bloecke**.
</Callout>
